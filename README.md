# Expense Tracker (C++)

## Description
This project is a simple Expense Tracker developed using C++. It helps users record and manage their daily expenses.

## Features
- Add Expense
- View Expenses
- Calculate Total Expense

## Technologies Used
- C++
- File Handling

## How to Run
1. Compile main.cpp
2. Run the program
3. Choose options from menu

## Project Structure
- main.cpp → Main program
- expenses.txt → Stores expense data
- README.md → Project details

## Note
This project uses file handling to store data permanently.
